# Sistema de Recomendação de Filmes - Paradigma Imperativo (C)

## Versão utilizada

GCC 13.2.0

## Compilação

```bash
gcc main.c -o recomendador
```

## Execução

```bash
./recomendador
```

## Dependências

Nenhuma dependência externa.
Apenas bibliotecas padrão da linguagem C:

* stdio.h
* string.h
* stdlib.h

## Observações

A implementação segue o paradigma imperativo utilizando:

* Structs
* Arrays
* Loops explícitos
* Funções para encapsular operações

A ordenação dos resultados é feita utilizando qsort().
